# 健康监测APP

一款面向有慢性病风险人群的苹果端健康监测APP。

## 功能特性

- **食物识图**：拍照或上传食物图片，分析健康影响
- **饮食记录**：自动记录每日饮食，追踪健康状况
- **健康分析**：健康评分、趋势分析、个性化建议
- **会员系统**：1天试用，之后付费使用

## 技术栈

- React Native + Expo
- TypeScript
- Firebase
- SQLite

## 安装运行

```bash
npm install
npx expo start
```

## 配置

1. 创建Firebase项目
2. 配置 `src/services/firebase.ts` 中的API密钥
3. 配置医学大模型API
